# 数理配图原则（清晰、规范、够用就好）

1. **服务题目**：这是给数学/物理老师做题目配图的，画面只放题目需要的元素，无关装饰一律不画。
2. **标注规范**：关键点用大写字母（A、B、C…），边长/角度按题意标注；直角处画直角符号（Angle 命令）；标注文字位置不压线、不出视窗。
3. **比例正确**：按题目数据取真实坐标（3-4-5 就画 3-4-5），视窗留 10%~20% 边距，图形居中。
4. **黑白可印**：默认深色细线为主，颜色只用于强调 1~2 处重点；导出后缩印成黑白也要能看清。
5. **公式用 LaTeX**：结论/公式文本用 LaTeX 渲染，保持默认黑色。
6. **克制**：同屏文本 ≤ 3 块；对象总数 ≤ 15。
7. **交互按需**：只有用户明确要求「滑杆 / 动点 / 动画 / 演示变化」时才做交互元素；普通配图一律画静态图。
